Дополнительная информация для задания 1.

Выход за пределы Docker-контейнера

Последовательность действий для выполнения: 
1. Загрузить образ alpine на локальную машину. 
2. Присвоить контейнеру имя “capability”.
3. Установить конфигурацию разрешающую все возможности(capabilities) внутри контейнера. 
4. Убрать возможность CHOWN. 
5. Запустить контейнер в режиме демона.

Дополнительная информация для задания 2.

Злоупотребление ресурсами в docker

Последовательность действий для выполнения: 
1. Загрузить образ nginx  на локальную машину. 
2. Установить конфигурацию проброса портов с 80 на 8080. 
3. Установить лимит оперативной память на 128M 
4. Указать ограничения cpu на 30%. 
5. Запустить контейнер в режиме демона.
